Explication des fonctions WinDev

HLitRecherche : permet de rechercher un enregistrement précis dans un fichier, en se basant sur une clé ou un champ spécifique.

HSupprime : sert à supprimer l'enregistrement actuellement sélectionné.

HTrouve : est utilisé juste après une recherche pour vérifier si un enregistrement a bien été trouvé.

HModifie : permet de sauvegarder les modifications apportées à un enregistrement.

HLitSuivant : permet de parcourir tous les enregistrements d’un fichier un par un, comme dans une boucle.
